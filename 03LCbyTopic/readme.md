### 做题攻略

+ 1、前250道（经典+必考）
+ 2、第2次不能解加入收藏夹（**没有提示，通过2次，移出**）
+ 3、尽量不用IDE，**给自己思考空间，不要打开题目标穄 **，
  + **刷了3遍以上没有达到理想目标，一定是方法出问题了，多总结**
+ 4、写好代码，先不提交，人工检查一下，自定义测试用例，检查边界
